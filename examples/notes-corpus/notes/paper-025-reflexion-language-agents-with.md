Just finished reading "Reflexion: Language Agents with Verbal Reinforcement Learning" (https://arxiv.org/abs/2303.11366).

This reframed how I think about the knowledge freshness problem in retrieval systems. The evaluation methodology is worth borrowing, separate from the results. Several ideas here transfer directly to agent pipeline design, even though that wasn't the paper's framing. The technique maps cleanly onto the notes-corpus ingestion pipeline I'm designing. I'll try the main heuristic on a real codebase next week to see if it generalises. I can see a direct application to how I structure context for long agent sessions.
