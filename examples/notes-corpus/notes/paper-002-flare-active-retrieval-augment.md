Just finished reading "FLARE: Active Retrieval Augmented Generation" (https://arxiv.org/abs/2305.06983).

The evaluation methodology is worth borrowing, separate from the results. The ablation tables are where the real insight lives — the abstract undersells the nuance. The failure cases section was more useful to me than the success cases. I can see a direct application to how I structure context for long agent sessions. The technique maps cleanly onto the notes-corpus ingestion pipeline I'm designing. The approach suggests a way to audit legacy code systematically rather than heuristically.
