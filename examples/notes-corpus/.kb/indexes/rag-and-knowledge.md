# RAG and Knowledge Retrieval

Implementation of retrieval-augmented generation: embeddings, vector search, hybrid pipelines, knowledge graphs, codebase summarization.

## When to enter

- You're implementing embedding-based retrieval for a corpus.
- You're choosing between dense retrieval, BM25, or hybrid approaches.
- You're summarizing a codebase or note corpus for agent navigation.
- You need to build or query a knowledge graph from unstructured notes.
