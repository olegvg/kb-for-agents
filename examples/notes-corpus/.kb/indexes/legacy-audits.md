# Legacy Audits

AI-assisted auditing of legacy codebases and test-driven development as the audit follow-on.

## When to enter

- You're starting an AI-assisted audit of an underdocumented or legacy codebase.
- You want to formalize undocumented behaviors found during an audit into tests.
- You're looking for incremental audit approaches that don't require a full one-shot session.
- You're using diff-based code editing to implement audit findings.
