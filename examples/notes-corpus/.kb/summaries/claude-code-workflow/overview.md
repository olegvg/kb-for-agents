# Claude Code Workflow — Overview

This domain covers the Claude Code environment specifically: its configuration mechanisms (CLAUDE.md, slash commands, keybindings, hooks), its plugin and MCP plugin authoring capabilities, and the operational patterns for keeping long Claude Code sessions productive.

The corpus includes tool notes from direct Claude Code practice (`tool-000`, `tool-002`, `tool-022`, `tool-004`, `tool-018`) and cost/efficiency notes (`tool-019`, `tool-026`, `tool-023`). These are distinct from general agent patterns — they are specific to the Claude Code host environment and its affordances.

## Leaves in this domain

- **[configuration](configuration.md)** — CLAUDE.md instructions, slash commands, keybindings, hook-driven automation.
- **[plugins-and-extensions](plugins-and-extensions.md)** — plugin authoring for Claude Code, MCP integration from the Claude Code side.
- **[cost-and-efficiency](cost-and-efficiency.md)** — prompt caching, cost management for long runs, PR review automation.
